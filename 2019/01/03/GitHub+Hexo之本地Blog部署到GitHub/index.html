GitHub+Hexo之本地Blog部署到GitHub

发表于 2019-01-03 | 更新于 2019-01-11 | 分类于 hexo
本文字数： 611 | 阅读时长 ≈ 1 分钟

前言

我们在上一篇文章已经把本地的hexo环境搭建好了，并且在本地成功预览，但是本地预览也只能我们能自己看到，其他人根本看不到也得不到学习和交流的效果，这篇文章我们说一下如何把本地Blog部署到GitHub上让更多的朋友来访问我们的博客。

步骤

在github上创建一个仓库

注意此仓库的名称格式是： 你的github用户名.github.io (其中用户名不区分大小写) 如下图

修改hexo文件配置

  1. 要想把本地blog部署到github，需要对本地myblog文件夹下的 _config.yml 文件增加配置 如图
  2. 打开_config.yml文件，拉到最后，找到 deploy 选项，进行配置:注意标记的地方都要有空格，其中， repo 填写你刚刚创建github的仓库的地址

部署到GitHub

  1. 打开myblog文件夹，调出Powershell终端，首先安装deployer插件cnpm i hexo-deployer-git -S
  2. 安装完毕后，通过 hexo g 生成静态文件
  3. 输入命令 hexo d 完成部署
  4. 再进入到github，进入io仓库，点击setting 进入设置，往下拉至 GitHub Pages 看到 liuzxPower.github.io，这时候只需要地址栏中输入 iuzxPower.github.io (输入自己的github名称.github.io) 即可访问部署成功的Blog
